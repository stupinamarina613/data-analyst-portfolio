## Проекты

### Проект 1: Комплексный анализ эффективности для федерального ретейлера
**Задача:** Провести всесторонний анализ операционной и маркетинговой деятельности для выявления точек роста.

*   **Что было сделано:**
    *   Проведен инцидент-анализ: систематизированы все происшествия, определены их причины и рассчитан финансовый ущерб
    *   Проанализирована эффективность маркетинговых активностей: создан каталог всех акций, проведен финанализ их окупаемости, сформулированы рекомендации для отдела маркетинга
    *   Разработан интерактивный дашборд в **Google Sheets** для мониторинга ключевых показателей с подробной инструкцией для пользователей

*   **Результат:** Выявлены основные источники финансовых потерь и неэффективные маркетинговые каналы. Внедренные рекомендации позволили сократить операционные затраты на 15% в следующем квартале.

*   **Инструменты:** `Google Sheets`, `SQL`, `Статистический анализ`

### Проект 2: Анализ пользовательского поведения в онлайн-кинотеатре
**Задача:** Разработать систему оценки успешности сервиса и найти причины падения просмотров ключевого контента.

*   **Что было сделано:**
    *   Построена система ключевых метрик: **Retention Rate** (удержание) и **LTV** (пожизненная ценность пользователя)
    *   Проведен детальный анализ "проседания" просмотров популярного сериала, выдвинуты и приоритизированы гипотезы, объясняющие падение

*   **Результат:** Менеджмент получил четкую систему для оценки здоровья продукта. Команде контента были предложены 3 основные гипотезы для немедленной проверки и корректировки стратегии.

*   **Инструменты:** `SQL`, `Python (Pandas)`, `Retention-анализ`, `Когортный анализ`

### Проект 3: Анализ продукта и монетизации мобильной игры "Zombie Revolution"
**Задача:** Оценить стабильность продукта, эффективность пользовательской монетизации и рекламных кампаний.

*   **Что было сделано:**
    *   Рассчитаны базовые метрики продукта: **DAU/MAU**, длительность и количество сессий
    *   Проведен анализ удержания пользователей (конверсии по дням жизни)
    *   Оценена монетизация: рассчитаны **ARPU** (средний доход на пользователя) и **ARPPU** (средний доход на платящего пользователя)
    *   Проанализирована окупаемость рекламных кампаний

*   **Результат:** Выявлены проблемные точки в воронке удержания новых пользователей. Даны рекомендации по оптимизации рекламного бюджета и точкам монетизации в игре.

*   **Инструменты:** `SQL`, `Python (Pandas, Matplotlib, Seaborn)`, `Продуктовые метрики`

### Дипломный проект: Оценка эффективности пуш-уведомлений в мобильном приложении (A/B тест)
**Задача:** Определить, какой формат уведомлений (новые пуш-уведомления vs. традиционные баннеры) эффективнее с точки зрения конверсии в покупку и среднего чека.

*   **Что было сделано:**
    *   Проведена полная подготовка и очистка данных о платежах и пользователях
    *   Проанализированы результаты A/B-теста с применением статистических критериев (**t-тест**, **критерий Манна-Уитни**)
    *   Выполнена сегментация по городам (Москва, Санкт-Петербург, регионы) для выявления географических особенностей
    *   Результаты визуализированы, а выводы оформлены в виде комплексного **Excel-калькулятора** для менеджмента

*   **Результат:** Определены сегменты (торговые точки и города), где новый формат уведомлений показал статистически значимый рост. Калькулятор позволяет бизнесу оценить финансовый эффект от масштабирования нововведения.

*   **Инструменты:** `Python (Pandas, SciPy, NumPy)`, `A/B тестирование`, `Статистика`, `Excel`, `Визуализация`
