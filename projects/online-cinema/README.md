# Анализ пользовательского поведения в онлайн-кинотеатре

## Задача
Разработать систему оценки успешности сервиса и выявить причины падения просмотров ключевого контента.

## Описание проекта
В рамках проекта была построена система метрик для мониторинга здоровья продукта и проведено исследование причин аномального падения просмотров популярного сериала.

## Методология

### 1. Построение системы метрик
- Рассчитан **Retention Rate** (показатель удержания) по когортам для анализа долгосрочной лояльности пользователей
- Определен **LTV** (Lifetime Value) для оценки пожизненной ценности пользователей
- Настроено регулярное отслеживание ключевых показателей активности

### 2. Исследование падения просмотров
- Проведен детальный анализ воронки просмотров целевого сериала
- Выдвинуты и приоритизированы гипотезы, объясняющие падение:
  1. **Технические проблемы** - сбои в работе плеера или рекомендательной системы
  2. **Контентные факторы** - несоответствие ожиданиям аудитории (сюжет, актерский состав)
  3. **Внешние факторы** - появление конкурентного контента на других платформах
- Проведена сегментация аудитории для выявления наиболее пострадавших групп пользователей

## Результаты
- Внедрена система мониторинга ключевых метрик продукта
- Выявлены 3 наиболее вероятные причины падения просмотров с обоснованием приоритетности проверки
- Сформированы конкретные рекомендации для команды контента:
  - Проверить технические метрики плеера за период падения
  - Проанализировать отзывы и рейтинги сериала
  - Исследовать активность в социальных сетях

## Инструменты
- **Анализ данных:** Excel (сложные формулы, сводные таблицы)
- **Методологии:** Retention-анализ, когортный анализ, построение воронок
- **Визуализация:** Диаграммы и графики в Excel

## Выводы
Проект демонстрирует способность проводить комплексный продуктовый анализ без использования специализированного ПО, используя глубокое понимание метрик и пользовательского поведения. Результаты позволили продукту оперативно среагировать на проблему и начать тестирование гипотез для восстановления показателей.
