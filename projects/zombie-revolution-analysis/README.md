# Анализ метрик и монетизации мобильной игры Zombie Revolution

## Задача
Провести комплексный анализ пользовательского поведения, оценить эффективность монетизации и маркетинговых активностей, разработать критерии выявления лояльных пользователей.

## Описание проекта
В рамках проекта проведен полный анализ жизненного цикла игры с июля 2022 по апрель 2023 года, включая исследование вирального роста, технических проблем и сегментации пользовательской базы для оптимизации бизнес-показателей.

## Методология

### 1. Анализ пользовательской базы и активности
- Проанализирована база из **3,101 регистрации** и **3,078 уникальных пользователей**
- Выявлен факт многократных регистраций (**23 пользователя**)
- Построена динамика пользователей по месяцам с выявлением **резкого скачка в марте 2023** и последующего спада
- Проанализировано **27,161 игровых сессий**, из которых **19,357 (71%)** длились более 5 минут

### 2. Исследование вовлеченности и реферальной системы
- Рассчитана **доля значимых сессий (71%)** - ключевой показатель удержания
- Проанализирована реферальная система: **7,933 отправленных приглашений** с **конверсией 35%** в установку
- Рассчитан **K-factor = 0.93** по формуле: 2.62 (среднее количество регистраций) × 0.35 (конверсия)
- Выявлена **чистая убыль 20 пользователей на когорту** при текущих показателях

### 3. Оценка маркетинговых активностей и монетизации
- Проанализирована эффективность мартовской акции:
  - **MAU вырос на 121.84%** по сравнению с февралем
  - **WAU увеличился на 66.5%**
  - **DAU вырос на 76.3%**
- Зафиксирован **рекордный рост выплат** по валюте, материалам и оружию
- Отмечено **незначительное снижение Sticky Factor** несмотря на общий рост активности

### 4. Выявление технических проблем
- Обнаружено **3,287 сессий (12%)** без времени окончания
- Проведена сегментация по платформам:
  - **iOS: 18.98% проблемных записей** (97.44% от всех проблем)
  - **Android: 0.8% проблемных записей**
- Сделан вывод о **приоритетности оптимизации iOS-клиента**

### 5. Сегментация лояльных пользователей
Протестированы 5 критериев лояльности:
1. **crit_invite** - пользователи, пригласившие ≥3 друзей с ≥1 регистрацией
2. **crit_1000** - пользователи с суммарными платежами >1000 рублей
3. **Комбинированный AND** - слишком жесткий критерий, недостаточно пользователей
4. **Комбинированный OR** - слишком широкий критерий
5. **Топ-100 по средним выплатам** - оптимальный баланс для выявления ценных пользователей

### 6. Дополнительные исследования
- Подтверждена гипотеза о **большей лояльности когорт ноября-декабря 2022**:
  - **3 часа 38 минут** vs **2 часа 57 минут** средняя длительность сессий
- Выявлены **сезонные колебания** - спад доли длинных сессий в декабре

## Ключевые выводы

### Положительные результаты:
- ✅ **Высокий показатель удержания** - 71% сессий длительнее 5 минут
- ✅ **Эффективная реферальная система** с конверсией 35%
- ✅ **Успешная мартовская акция** с ростом метрик на 66-121%
- ✅ **Качественные когорты** ноября-декабря 2022 с повышенной лояльностью

### Проблемные зоны:
- ⚠️ **K-factor 0.93** требует оптимизации для устойчивого роста
- ⚠️ **Технические проблемы на iOS** влияют на 12% сессий
- ⚠️ **Сезонные спады** в декабре и после мартовского пика

## Рекомендации

### Высокий приоритет:
1. **Оптимизировать iOS-клиент** для устранения технических проблем
2. **Увеличить конверсию рефералов до 50%** для достижения K-factor > 1
3. **Внедрить сегмент Топ-100 пользователей** в программы лояльности

### Средний приоритет:
4. **Повторить успешные механики марта 2023** с акцентом на удержание
5. **Улучшить первые 5 минут геймплея** для увеличения retention
6. **Разработать сезонные стратегии** для сглаживания спадов

## Инструменты и технологии
- **Анализ данных:** SQL (расчет метрик, когортный анализ, сегментация)
- **Визуализация:** PowerPoint (дашборды и презентация результатов)
- **Ключевые метрики:** DAU/WAU/MAU, Retention, K-factor, ARPU, LTV
- **Методологии:** Когортный анализ, виральный маркетинг, RFM-сегментация
