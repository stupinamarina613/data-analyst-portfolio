# A/B тестирование пуш-уведомлений vs баннеров в мобильном приложении

## Задача
Оценить эффективность нового типа уведомлений (пуш-уведомления) по сравнению с традиционными баннерами в мобильном приложении розничного ретейлера. Проверить гипотезу о повышении конверсии в покупку и среднего чека при использовании пуш-уведомлений.

## Описание проекта
В течение 3 месяцев проводилось A/B тестирование в 15 городах России с участием 55,356 клиентов. Контрольная группа получала баннеры в приложении, тестовая группа - пуш-уведомления. Анализ включал статистическую проверку гипотез, сегментацию по городам и создание бизнес-калькулятора для принятия решения о масштабировании.

## Ключевые результаты
- **+7.25% к среднему чеку** в тестовой группе (2,201.52 руб. → 2,361.12 руб.)
- **+4.35% к конверсии** в покупку (49.98% → 54.33%)
- **Статистическая значимость** подтверждена t-тестом и критерием Манна-Уитни (p-value < 0.0001)
- **Выявлена геосегментация**: максимальный эффект в Москве и Санкт-Петербурге
- **Создан Excel-калькулятор** для оценки экономического эффекта масштабирования

- Исключение проблемных торговых точек (15 точек с некорректными данными)
